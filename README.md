# DiscordWebhooks
В Discord заводите входящий вебхук на нужный канал (или несколько), вставляете URL в админку Flute и отмечаете, какие события с сайта туда слать. Один канал Discord может получать только «регистрации и донаты», другой — весь поток включая новости, если такие события подключены.  Из коробки доступны типы вроде: новый пользователь, вход, подтверждённ
